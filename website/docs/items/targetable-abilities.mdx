## Making abilities targetable {#making-targetable}

Just implement the `IAbilityTargetable` interface in your ability's class:

```csharp title="MyTargetableAbility.cs"
// highlight-next-line
public class MyTargetableAbility : CustomAbility, IAbilityTargetable
{
    // highlight-next-line
    public PlayfieldObject FindTarget() { /* ... */ }
}
```

`FindTarget` determines the closest target, that the ability can be used on. An ability indicator will be displayed over that object. If the ability cannot be used right now, you should return `null`.

Then, you can use `CurrentTarget` in any of the methods:

```csharp
    public override void OnPressed()
    {
        if (CurrentTarget is null)
        {
            gc.audioHandler.Play(Owner, "CantDo");
            return;
        }
        /* ... */
    }
```
