# Введение

RogueLibs - библиотека для моддинга Streets of Rogue.

Код игры довольно плох, так что если вы попытаетесь сделать мод на игру сами, то у вас выйдет слишком много патчей! Пусть RogueLibs займётся патчами, а **вы просто сфокусируйтесь на контенте!**

Зацените [гайд по установке](user/installation)!

Тут также есть [руководство для разработчиков плагинов](getting-started)!
