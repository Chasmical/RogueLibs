## `IDoUpdate` и `IDoFixedUpdate` {#update-interfaces}

Если вы хотите обновлять свои предметы/особенности/разблокировки с `Update` или `FixedUpdate` Unity, реализуйте эти интерфейсы:

<Tabs
    defaultValue="update"
    values={[
        {label:'IDoUpdate', value:'update'},
        {label:'IDoFixedUpdate', value:'fixedupdate'},
    ]}>
<TabItem value="update">

```csharp
// highlight-next-line
public class MyCustomItem : CustomItem, IDoUpdate
{
    // highlight-start
    public void Update()
    {
        /* ... */
    }
    // highlight-end
}
```

</TabItem>
<TabItem value="fixedupdate">

```csharp
// highlight-next-line
public class MyCustomItem : CustomItem, IDoFixedUpdate
{
    // highlight-start
    public void FixedUpdate()
    {
        /* ... */
    }
    // highlight-end
}
```

</TabItem>
</Tabs>
