## Делаем способности заряжаемыми {#making-chargeable}

Просто реализуйте интерфейс `IAbilityChargeable` в классе вашей способности:

```csharp title="MyChargeableAbility.cs"
// highlight-next-line
public class MyChargeableAbility : CustomAbility, IAbilityChargeable
{
    // highlight-start
    public void OnHeld(AbilityHeldArgs e) { /* ... */ }
    public void OnReleased(AbilityReleasedArgs e) { /* ... */ }
    // highlight-end
}
```

`OnHeld` вызывается каждый кадр *(вроде как?)* на котором кнопка специальной способности удерживается. `OnReleased` вызывается на кадре, на котором кнопка специальной способности отпускается. Используйте их вместе с `OnPressed` для зарядки вашей способности и всякое такое.

:::caution
Метод `AbilityHeldArgs.Interrupt()` ещё в процессе написания.
:::
