IDoUpdate and IDoFixedUpdate#

If you want to update your items/traits/unlocks with Unity's Update or FixedUpdate, implement these interfaces:

  • IDoUpdate
  • IDoFixedUpdate
public class MyCustomItem : CustomItem, IDoUpdate{    public void Update()    {        ...    }}
public class MyCustomItem : CustomItem, IDoFixedUpdate{    public void FixedUpdate()    {        ...    }}
