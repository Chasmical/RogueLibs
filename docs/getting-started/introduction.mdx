## Why should you use RogueLibs?

- <span style={{ fontSize: "1.1em" }}>

  Because it's really easy to use! Just check out some examples in [here](https://github.com/SugarBarrel/RogueLibs/tree/main/RogueLibsCore.Test/Tests).

  </span>

  </span>
